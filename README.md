# The Next Level: Prediciendo el Futuro del Gaming con IA 🎮📊

Este proyecto es el resultado de la integración de conocimientos de diversas materias del semestre en la carrera de **Ciencia de Datos e Inteligencia Artificial** en **CUGDL**. El objetivo es analizar el mercado global de videojuegos y predecir tendencias de éxito utilizando Machine Learning.

## 🚀 Materias Integradas
*   **Bases de Datos:** Diseño, normalización (3FN) e implementación del esquema en SQL.
*   **Machine Learning:** Implementación de 5 modelos de predicción y clasificación.
*   **Matemáticas Aplicadas:** Fundamentación teórica y desarrollo paso a paso de los algoritmos[cite: 1].
*   **Complejidad Algorítmica:** Análisis de eficiencia (Big O) comparando implementaciones manuales vs. librerías[cite: 1].
*   **Programación:** Estructuración de código limpio, modular y colaborativo mediante GitHub[cite: 1].

## 📂 Estructura del Repositorio
*   `/data`: Contiene el dataset original de Video Game Sales (Kaggle) y versiones procesadas[cite: 1].
*   `/notebooks`: Notebooks de Google Colab con el EDA y los modelos de ML[cite: 1].
*   `/sql`: Scripts de creación de tablas y carga de datos.
*   `/docs`: Documentación técnica, DER y póster final[cite: 1].

## 🛠️ Tecnologías Utilizadas
*   **Lenguajes:** Python (Pandas, Scikit-Learn), SQL.
*   **Herramientas:** Google Colab (Entorno de cómputo), dbdiagram.io (Modelado), Git/GitHub (Versicionamiento)[cite: 1].
